## 🎤 Voice Recognition

# ⚡ How It Works

```text
1. User clicks the microphone button
2. Browser listens to speech
3. Speech converts into text
4. App detects selected language
5. Translation API processes text
6. Translated result appears instantly
7. User can copy, listen, or download output
```
